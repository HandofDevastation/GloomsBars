# Gloom's Bars

A **pure appearance layer** for World of Warcraft's built-in action bars — rounded / 3:2
icons, restyled keybind and count text, and **shape-matched proc glows and cooldown sweeps**
that actually follow the icon's shape instead of tracing a square around it.

Gloom's Bars never replaces Blizzard's secure action buttons. Edit Mode keeps owning
layout (position, scale, rows, spacing); this addon owns only the *look*. That keeps it
lightweight and free of the taint / combat-lockdown machinery of full bar replacements.

Bespoke for the **Hand of Devastation** guild. Sibling to
[Gloom's Auras](https://github.com/HandofDevastation/GloomsAuras) and Gloom's Build Barn.

## Install (WoWUp)

1. In WoWUp: **Get Addons → Install from URL**
2. Paste this repository's URL
3. WoWUp installs the latest GitHub Release and auto-updates from there.

## Use

`/gb` in-game for commands.

- Target: WoW **Midnight** retail (Interface `120007`)
